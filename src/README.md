Projeto Funcionários

Projeto desenvolvido em Java com o objetivo de gerenciar uma lista de funcionários.

O sistema realiza operações como cadastro, remoção, ordenação e cálculos sobre os dados dos funcionários.

## Funcionalidades
* Inserção de funcionários
* Remoção de funcionário específico
* Listagem com formatação de data e salário
* Aplicação de aumento salarial
* Filtro por mês de aniversário
* Identificação do funcionário mais velho
* Ordenação alfabética
* Cálculo do total de salários
* Cálculo de salários mínimos por funcionário

## Tecnologias utilizadas
* Java
* Eclipse IDE
* Git e GitHub
